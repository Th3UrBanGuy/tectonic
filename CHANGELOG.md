# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [2.0.0] - 2026-01-23

### Added
- Neon PostgreSQL database integration
- JWT authentication system
- Vercel Functions API layer
- Protected admin routes
- User management system
- Comprehensive SDP documentation
- Security documentation
- Database schema with 7 tables
- Seed data with default content

### Changed
- Migrated from localStorage to database storage
- Updated content storage service to hybrid mode
- Improved login page design

### Security
- Added bcrypt password hashing
- Implemented JWT token verification
- Added role-based access control
- Secured API endpoints with middleware
