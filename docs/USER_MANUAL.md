## Managing Content

### Content Types

| Type | Description | Where Used |
|------|-------------|------------|
| Wings | Company divisions | Home, Wings page |
| Team | Team members | Company page |
| Timeline | Milestones | Company page |
| Partnerships | Partners | Home page |
| Projects | Portfolio items | Portfolio page |
| Tech Stack | Technologies | Innovation page |
| Roadmap | Future plans | Innovation page |

### Editing Content

1. Go to **Content** in sidebar
2. Select content type tab
3. Click on item to edit
4. Make changes
5. Click **Save**

### Adding New Items

1. Go to relevant content section
2. Click **Add New**
3. Fill in all fields
4. Click **Save**

### Deleting Items

1. Find the item
2. Click the delete icon (ğŸ—‘ï¸)

## User Management

### Adding New Admin

1. Go to **Users**
2. Click **Add User**
3. Enter:
   - Email
   - Password
   - Name
   - Role (admin)
4. Click **Create**

### Changing Password

1. Go to **Settings**
2. Click **Change Password**
3. Enter current password
4. Enter new password twice
5. Click **Update**

## Troubleshooting

### Can't Login

1. Check email is correct
2. Check password (case-sensitive)
3. Clear browser cache
4. Try incognito mode

### Changes Not Saving

1. Check internet connection
2. Refresh the page
3. Try again
4. Contact support if persists

### Page Not Loading

1. Check internet connection
2. Clear browser cache
3. Try different browser
4. Check if site is in maintenance mode
